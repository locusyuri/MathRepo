# Illustration Prompt Crafting（绘图提示词编写）

当你需要根据笔记内容生成物理/数学教学插图提示词时使用本技能。

## 1. 前置准备

读取用户 `.typ` 文件中的相关内容，确认：

- **物理模型**：点电荷、载流线圈、带电球体等
- **数学公式**：场强、电势、磁场等的函数表达式
- **参数值**：半径 $R$、电流 $I$、电荷 $q$ 等
- **对称性**：球对称、柱对称、平面对称、轴对称
- **坐标/符号**：笔记中使用的变量名和坐标系

## 2. 布局设计

**默认规则**：除非用户特殊说明，否则所有插图都采用**左右两面板布局**（左 = 模型示意图，右 = 相关函数曲线）。

| 概念类型 | 推荐布局 |
|---------|---------|
| 单一模型（电场线、几何示意） | 左右两面板：左 = 模型示意图，右 = 相关函数曲线 |
| 模型 + 函数曲线 | 左右两面板：左 = 模型，右 = 函数曲线 |
| 模型 + 多条函数曲线 | 左右两面板：左 = 模型，右 = 合并的函数曲线图（多曲线在同一坐标系） |
| 纯函数比较 | 左右两面板：左 = 模型示意图，右 = 多条曲线叠在同一坐标系中，不同颜色/线型区分 |
| 多区域场（球壳内/外） | 左右两面板：左 = 模型示意图，右 = 分段函数曲线图（合并显示 E 和 V） |

多个相关函数（如 E-r 和 V-r）应尽量**画在同一张图中**，而不是分开成多个面板。

| 新概念类型 | 推荐布局 |
|-----------|---------|
| 磁场模型（立体环绕场） | 左右两面板：左 = **3D 侧视图**（立体感，闭环），右 = 函数曲线 |

### 磁场的"3D vs 2D"视角选择

磁场模型图需要特别关注视角选择，因为磁感线是空间的闭合环：

| 视角 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **3D 侧视图**（导线竖直，磁感线环投影为椭圆） | 无限长直导线、同轴电缆等 | 同时显示导线和环绕的磁场线，最直观 | 椭圆可能被误认为磁场本身形状 |
| **2D 顶视图**（沿导线轴向看） | 纯磁场线拓扑示意 | 显示真正的圆环形 | 无法显示导线和电流方向 |
| **3D 轴测图** | 有限长导线、电流环 | 立体感强，适合演示 | 结构较复杂 |

**推荐默认**：**3D 侧视图**（与 `magnetic_field_loop.py` 风格一致），磁场线为环绕导线的同心圆环（投影为椭圆），B 矢量沿环的切向。

## 3. 数学验证（关键步骤）

在编写提示词前，必须逐一验证每个公式和数值。

### 选取数值范围的原则

选取的数值必须**体现函数的整体特征**，而不是局部细节：

| 函数类型 | 应选取的范围 | 反例（错误） |
|---------|-------------|-------------|
| $y = 1/x$（反比） | $x \in [0.5, 5]$，展示快速衰减 | $x \in [1000, 1005]$，曲线近乎水平 |
| $y = 1/x^2$（反平方） | $x \in [0.5, 4]$，展示骤降趋势 | $x \in [100, 105]$ |
| $y = e^{-x}$（指数衰减） | $x \in [0, 5]$，展示从 1→0 全过程 | $x \in [1000, 1005]$ |
| 分段函数（球壳） | 覆盖 $r < R$、$r = R$、$r > R$ 三段 | 只取 $r > R$ |
| 振荡函数 | 至少覆盖 2-3 个完整周期 | 不到半个周期 |

### 验证项

1. **函数表达式**：从 `.typ` 文件中读取，确认无误
2. **数检验证**：在选取的范围内计算 2-3 个特征点的值
3. **量纲检查**：确认每个物理量的单位正确
4. **渐近行为**：$r \to 0$、$r \to \infty$ 时的极限行为
5. **一致性检查**：如 $E = -dV/dr$、$\nabla \cdot \mathbf{B} = 0$ 等

验证表输出示例：

```
## 验证

### 参数
- 使用通用符号：k、q、R 等，不标注具体数值

### 数值检查
| r (m) | E = kq/r² (N/C) | V = kq/r (V) |
|-------|-----------------|--------------|
| 0.5   | 计算值示例      | 计算值示例   |
| 1.0   | 计算值示例      | 计算值示例   |
| 2.0   | 计算值示例      | 计算值示例   |

### 量纲检查
- [E] = N/C ✓
- [V] = J/C = V ✓

### 渐近行为
- r → 0: E → ∞, V → ∞ ✓
- r → ∞: E → 0, V → 0 ✓

### 一致性检查
- E = -dV/dr: kq/r² = kq/r² ✓
```

## 4. 编写提示词

### 结构模板

```
【整体描述】
[透明背景、风格、配色方案、字体]

【面板1：子图标题】
[具体内容：对象、颜色、数量、标注]

【面板2：子图标题】
[具体内容：坐标系、曲线、标注、图例]

面板之间用适当留白分隔，不使用实线分割。
```

### 必须包含的要素

- **坐标轴标签带单位**：`Distance r (m)`、`E (N/C)`、`V (V)`、`B (T)`
- **函数公式在图上**：在曲线附近用粗体显示公式，如 `E = kq / r²`（不标注具体参数值，如 k、q、R 等）
- **多个特征点标注**：至少 2 个关键点的数值
- **网格线**：浅灰色，适当间距
- **图例**：颜色对应说明
- **子图小标题**：如 `...`、`...`
- **不画大标题**

### 配色规范

| 元素 | 颜色 | 用途 |
|------|------|------|
| 正电荷/源 | 红色 | 点电荷、从正电荷出发的电场线 |
| 负电荷/汇 | 蓝色 | 负电荷、指向负电荷的电场线 |
| 电场强度 E | 蓝色 | 电场矢量、E 曲线 |
| 磁场 B | 绿色 | 磁场矢量、B 曲线 |
| 电势 V | 绿色或橙色 | V 曲线（与 E 在同一坐标系中时用不同颜色区分） |
| 电流/导线 | 红色 | 载流导线、电流方向箭头 |
| 网格/渐近线 | 浅灰色 | 辅助线 |
| 中性元素 | 黑色 | 坐标轴、标签、文字、图框 |

## 5. 常见物理模型图模板

### 5.1 点电荷电场

**模型图**（单面板）：
- 红色实心圆点 `+q`，位于画面中央
- 12 条红色径向电场线向外辐射，带箭头
- 若干测试点处的蓝色 E 矢量箭头
- 图例

**函数图**（单面板，合并 E-r 和 V-r）：
- 同一坐标系中画两条曲线，用不同颜色区分
- 蓝色曲线：$E = kq/r^2$（用左侧 y 轴）
- 绿色/橙色曲线：$V = kq/r$（用右侧 y 轴；若量级相近也可共用同一 y 轴）
- 在 $r \to \infty$ 处画水平虚线渐近线 $V \to 0$
- 标注关键数值点
- 选取范围 $r \in [0.5, 4]$ 体现从快速衰减到趋于平缓的全过程
- 不标注具体参数值（k、q 等）

### 5.2 载流圆环轴线磁场

**模型图**（单面板）：
- xy 平面内的蓝色圆环，标注半径 $R$
- 红色电流方向箭头沿圆环
- z 轴上若干个点处的绿色 B 矢量箭头，标注 $B_z(z)$
- 标注 $z$、$I$

**函数图**（单面板）：
- 绿色曲线：$B_z(z) = \dfrac{\mu_0 I R^2}{2(R^2+z^2)^{3/2}}$
- 标注 $z=0$ 处 $B = \mu_0 I/(2R)$
- 标注 $z \gg R$ 区域 $B \propto 1/|z|^3$ 的衰减趋势
- 选取范围 $z \in [0, 5R]$ 体现从峰值到衰减的全过程

### 5.3 均匀带电球壳

**函数图**（单面板，合并 E-r 和 V-r）：
- 蓝色曲线：$E(r)$ 分段函数
  - $r < R$: $E = 0$（水平线）
  - $r \ge R$: $E = kQ/r^2$（递减曲线）
- 绿色曲线：$V(r)$ 分段函数
  - $r \le R$: $V = kQ/R$（水平线）
  - $r \ge R$: $V = kQ/r$（递减曲线）
- 在 $r = R$ 处画竖直虚线标注分界点
- 覆盖 $r \in [0, 4R]$ 展示全部三段行为

### 5.4 均匀带电球体

**函数图**（单面板，合并 E-r 和 V-r）：
- 蓝色曲线：$E(r)$
  - $r < R$: $E \propto r$（线性增长）
  - $r \ge R$: $E = kQ/r^2$（递减）
- 绿色曲线：$V(r)$
  - $r \le R$: $V = \dfrac{kQ}{2R}\left(3 - \dfrac{r^2}{R^2}\right)$（抛物线）
  - $r \ge R$: $V = kQ/r$（递减）
- 在 $r = R$ 处画竖直虚线
- 覆盖 $r \in [0, 4R]$

### 5.5 平行板电容器

**模型图**（单面板）：
- 两块平行板，带 `+Q` 和 `-Q`
- 板间均匀电场线（平行等距蓝色箭头），板外无场

**函数图**（单面板，合并 E-x 和 V-x）：
- 蓝色曲线：$E(x)$ — 板间为常数 $E = \sigma/\epsilon_0$，板外为 0
- 绿色曲线：$V(x)$ — 板间线性下降，板外常数
- 覆盖 $x$ 从一块板外侧到另一块板外侧的全范围

### 5.6 电偶极子

**模型图**（单面板）：
- 一对 `+q`（红色）和 `-q`（蓝色），间距 $d$
- 电场线从正电荷出发到负电荷终止（弯曲的弧线）
- 标注偶极矩 $\mathbf{p} = q\mathbf{d}$

**函数图**（单面板）：
- 画一条沿偶极子轴线的 $E(x)$ 或 $V(x)$ 曲线
- 标注 $x \to \pm\infty$ 时场强趋于 0

## 5.7 磁场模型与电场模型的关键差异

在编写磁场模型提示词时，必须注意以下本质区别：

| 对比项 | **电场（点电荷）** | **磁场（载流直导线）** |
|-------|------------------|---------------------|
| 场线形态 | **径向辐射**（从正电荷发出→负电荷终止） | **同心闭合圆环**（环绕导线，无始无终） |
| 场线方向 | 正电荷向外 / 指向负电荷 | 右手定则（电流向上 → 逆时针环绕） |
| 衰减规律 | $E \propto 1/r^2$（反平方） | $B \propto 1/\rho$（反比，衰减更慢） |
| 标量势 | $V$ 存在（$E = -\nabla V$） | **无标量势**（在电流区 $\nabla\times B \neq 0$） |
| 右面板内容 | $E$-$r$ + $V$-$r$ 两条曲线 | **只有 $B$ 一条曲线**（或可选加磁通量示意） |
| 一致性检查 | $E = -dV/dr$ | $\oint B\cdot dl = \mu_0 I_\text{enc}$ |

### 5.8 无限长直导线的磁场

**模型图**（左面板，3D 侧视图）：
- 画面中央一条竖直的灰色圆柱体（导线），沿 z 轴放置
- 导线内部一个红色箭头标注电流方向：竖直向上，标签 I
- 环绕导线的 4 个同心圆环（深绿色虚线或实线），半径递增，圆心在导线轴线上
- 每个圆环上带方向箭头（逆时针方向），表示磁感线方向（右手定则：电流向上 → 逆时针环绕）
- 在 ρ 处（右方某点）画一个绿色箭头 B，沿切向（逆时针方向），标签 bold(B)(rho)
- 标注从导线轴心到该点的水平虚线，带双向箭头和标签 rho
- 图例：红色箭头 = Current I，绿色圆环 = Magnetic field lines，绿色箭头 = B
- 画面简洁，突出"环绕"和"闭合"的磁场特征
- **说明**：这是 3D 侧视图，导线竖直，磁场线是环绕导线的同心圆环（投影为椭圆），B 矢量沿切向。与 `magnetic_field_loop.py` 的画法一致，但那是单个电流环的轴线场，这是无限长直导线的环绕场。
- **导线表示**：用灰色圆柱体（非纯直线）是为了体现 3D 立体感，避免被误认为 2D 平面内的线段。圆柱体的粗细应适中，不遮挡磁场线。

**函数图**（右面板）：
- 横轴：Radial Distance ρ (m)，范围 [0.5, 5]
- 纵轴：Magnetic Field B(ρ) (T)
- 绿色实线曲线：B(ρ) = μ₀I / (2πρ)，从 ρ=0.5 到 ρ=5
- 在曲线旁标注公式：B = μ₀I / (2πρ)（粗体，绿色）
- 标注 2-3 个特征点：
  - (ρ₁, B₁) 近处某点
  - (ρ₂, B₂) 远处某点
  - ρ → ∞ 处画水平虚线渐近线 B → 0
- 浅灰色网格线（虚线），适当密度
- 图例：绿色实线 = B(ρ) ∝ 1/ρ

**说明**：该模型没有标量势 V，因此右面板只画 B(ρ) 单条曲线。磁场随距离衰减比点电荷电场慢（1/ρ vs 1/ρ²），这是磁场的特征。

## 5.13 磁场模型的一致性检查模板

在验证磁场模型时，使用以下一致性检查项：

````
### 一致性检查
- 安培环路定理：∮B·dl = μ₀I_enc
  验证：B(ρ)·2πρ = (μ₀I/(2πρ))·2πρ = μ₀I ✓
- 高斯磁定律：∇B·dS = 0（磁感线闭合）✓
- 矢势关系：B = ∇×A（在库仑规范下）✓
- 对比静电场：B(ρ) = μ₀I/(2πρ) E(ρ) = λ/(2πε₀ρ)（数学形式完全平行）✓
```
## 6. 输出格式

每次输出包含三部分：

1. **完整绘图提示词**（中文描述，标注用英文）
2. **数学验证表**（数值、量纲、渐近行为、一致性）
3. **布局说明**（面板结构、比例说明）

## 7. 重要规则

1. **绝不猜测公式** — 从 `.typ` 文件读取，如不在文件中则从第一性原理推导
2. **数值范围体现函数特征** — 选取能展示完整变化趋势的范围，不取局部平坦段
3. **多条相关函数优先合并在同一图中** — 用不同颜色/线型区分，配图例
4. **不需要大标题** — 每个子图有自己的小标题即可
5. **公式显示在图上** — 在曲线附近标注函数表达式，但不标注具体参数值（k、q、R 等）
6. **提示词用中文，图片标注用英文** — 适配中文 AI 工具，匹配仓库英文标注习惯
7. **具体量化** — 说明线的数量、颜色、箭头位置等视觉细节
