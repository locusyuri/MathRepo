# Oscillation and Wave Illustration Templates（振动与波动插图模板）

本子技能涵盖振动与波动学的插图模板，用于生成阻尼振动、共振、波动传播等主题的模型示意图与函数曲线图。

---

## 与电磁学图的本质区别

振动波动图与电磁场图有根本性不同：

| 对比项 | **电磁场图** | **振动波动图** |
|-------|-------------|---------------|
| 核心变量 | 空间分布 $f(r)$ | 时间演化 $f(t)$ 或频率响应 $A(\omega)$ |
| 左面板内容 | 场线分布（静态） | 物理模型的运动状态快照 |
| 右面板内容 | 场强 vs 位置 | 位移 vs 时间 / 振幅 vs 频率 |
| 关键标注 | 场强、电势、磁通 | 振幅、频率、相位、周期、节点、波腹 |
| 典型布局 | 左右两面板 | 左右两面板 或 上下多面板（简正模） |

## 振动模型模板

### 阻尼振动的三种状态

**模型图**（左面板）：
三个竖直排列的弹簧振子模型（从上到下：欠阻尼、临界阻尼、过阻尼），每个模型包含：
- 顶部固定点（灰色方块）
- 弹簧（灰色螺旋线）
- 质量块（蓝色方块，标注 m）
- 阻尼器（仅在欠阻尼和临界阻尼中可见，灰色活塞装置，标注 b）
- 平衡位置用水平虚线标注
- 每个模型右侧画一个双向箭头表示运动方向
- 三个模型之间用水平虚线分隔
- 每个模型下方标注：欠阻尼 (Underdamped) β < ω₀、临界阻尼 (Critically damped) β = ω₀、过阻尼 (Overdamped) β > ω₀
- 图例：弹簧 = Spring (k)，阻尼器 = Damper (b)，质量 = Mass (m)

**函数图**（右面板）：
- 横轴：Time t (s)，范围 [0, 6T₀]，其中 T₀ = 2π/ω₀
- 纵轴：Displacement x(t) / A（归一化位移）
- 三条曲线在同一坐标系中：
  - 蓝色实线（欠阻尼 β < ω₀）：x(t) = A e^(-βt) cos(ω₁t)，振幅指数衰减的振荡曲线。画蓝色虚线包络线 ±A e^(-βt)。标注 ω₁ = √(ω₀² - β²)
  - 绿色实线（临界阻尼 β = ω₀）：x(t) = A(1 + ω₀t) e^(-ω₀t)，最快回到零的非振荡曲线
  - 橙色实线（过阻尼 β > ω₀）：x(t) = A(e^(-r₁t) + e^(-r₂t))/2，缓慢回到零的非振荡曲线
- 在曲线旁标注公式（粗体）
- 标注特征点：t=0 处 x=A（三条曲线起点相同），包络线衰减到 A/e 处画水平虚线标注时间常数 τ = 1/β
- 浅灰色网格线（虚线），适当密度
- 图例：蓝色实线 = Underdamped，绿色实线 = Critically damped，橙色实线 = Overdamped，虚线 = Envelope ±Ae^(-βt)
- 子图小标题：(b) Displacement vs. Time for Three Damping Regimes

### 共振曲线

**模型图**（左面板）：
- 左侧：弹簧振子模型（固定点 + 弹簧 k + 质量 m + 阻尼器 b），质量块画在平衡位置
- 右侧：旋转手柄或曲柄机构，表示外加驱动力 F₀cos(ωt)，标注 F₀ 和 ω
- 弹簧和阻尼器用箭头表示力的方向
- 标注：m（质量）、k（弹簧刚度）、b（阻尼系数）、F₀（驱动力幅值）、ω（驱动频率）
- 图例：弹簧 = Spring (k)，阻尼器 = Damper (b)，驱动力 = Driving force F₀cos(ωt)

**函数图**（右面板）：
- 横轴：Driving frequency ω/ω₀（归一化），范围 [0, 2.5]
- 纵轴：Amplitude A(ω)/A_max（归一化），范围 [0, 1.2]
- 四条曲线在同一坐标系中（对应不同阻尼比 β/ω₀）：
  - 蓝色实线（β/ω₀ = 0.05，弱阻尼）：尖锐高峰，峰值接近 ω/ω₀ = 1
  - 绿色实线（β/ω₀ = 0.1，中等阻尼）：较宽高峰，峰值略低于 1
  - 橙色实线（β/ω₀ = 0.3，较宽阻尼）：宽平峰，峰值明显低于 1
  - 灰色虚线（β/ω₀ = 0，无阻尼参考）：在 ω/ω₀ = 1 处画垂直虚线表示 A → ∞
- 标注：ω/ω₀ = 1 处画垂直虚线标注 Natural frequency ω₀。每条曲线峰值处画空心圆点，标注 Resonance frequency ω_res = ω₀√(1 - 2β²/ω₀²)
- 在 ω/ω₀ = 1 附近画水平双向箭头标注 FWHM（半高宽）Δω = 2β
- 浅灰色网格线（虚线），适当密度
- 图例：蓝色 = β/ω₀=0.05，绿色 = β/ω₀=0.1，橙色 = β/ω₀=0.3，灰色虚线 = Undamped (β→0)
- 子图小标题：(b) Resonance Curves — Amplitude vs. Driving Frequency

## 波动模型模板

### 弦的简正模

**布局**：上下三面板垂直排列（不是左右两面板），三个子图分别对应 n=1, 2, 3。

**面板1 — 第一简正模 n=1（Fundamental Mode）**：
- 横轴：Position x/L（归一化弦长），范围 [0, 1]
- 纵轴：Displacement u(x)/A₁
- 蓝色实线曲线：u₁(x) = A₁ sin(πx/L)，半个正弦波，两端为节点，中间为波腹
- 在 x=0 和 x=L 处画红色圆点标注节点 Node
- 在 x=L/2 处画红色虚线标注波腹 Antinode，双向箭头标注振幅 A₁
- 标注：λ₁ = 2L，f₁ = v/(2L)
- 浅灰色网格线
- 子图小标题：(a) n = 1 — Fundamental Mode (f₁)

**面板2 — 第二简正模 n=2（Second Harmonic）**：
- 绿色实线曲线：u₂(x) = A₂ sin(2πx/L)，一个完整正弦波，两端 + 中间共三个节点
- 在 x=0、x=L/2、x=L 处画红色圆点标注节点
- 在 x=L/4 和 x=3L/4 处画红色虚线标注波腹
- 标注：λ₂ = L，f₂ = 2f₁
- 子图小标题：(b) n = 2 — Second Harmonic (f₂ = 2f₁)

**面板3 — 第三简正模 n=3（Third Harmonic）**：
- 橙色实线曲线：u₃(x) = A₃ sin(3πx/L)，一个半正弦波，共四个节点
- 在 x=0、x=L/3、x=2L/3、x=L 处画红色圆点标注节点
- 在 x=L/6、x=L/2、x=5L/6 处画红色虚线标注波腹
- 标注：λ₃ = 2L/3，f₃ = 3f₁
- 子图小标题：(c) n = 3 — Third Harmonic (f₃ = 3f₁)

三个面板的纵轴范围一致，便于比较振幅。所有面板共享一个图例：实线 = Displacement uₙ(x)，红色圆点 = Node (u=0)，虚线 = Antinode。

## 振动波动模型的一致性检查

### 阻尼振动

```
- 特征方程：r² + 2βr + ω₀² = 0，根 r = -β ± √(β² - ω₀²)
- 欠阻尼：β < ω₀，ω₁ = √(ω₀² - β²)，振荡衰减
- 临界阻尼：β = ω₀，重根 r = -ω₀，最快非振荡衰减
- 过阻尼：β > ω₀，两个不等实根，缓慢非振荡衰减
- 能量耗散：dE/dt = -bv² < 0
```

### 共振

```
- 振幅公式：A(ω) = (F₀/m) / √((ω₀²-ω²)² + (2βω)²)
- 共振频率：ω_res = √(ω₀² - 2β²)
- 品质因子：Q = ω₀/(2β)
- 半高宽：FWHM = 2β
- β → 0 时，A(ω₀) → ∞
```

### 弦的简正模

```
- 波动方程：∂²u/∂t² = v² ∂²u/∂x²
- 简正模：uₙ(x,t) = Aₙ sin(nπx/L) cos(ωₙt)
- 频率：fₙ = nv/(2L) = nf₁
- 波长：λₙ = 2L/n
- 节点数 = n+1，波腹数 = n
- 边界条件：uₙ(0) = uₙ(L) = 0
- 正交性：∫₀ᴸ sin(mπx/L) sin(nπx/L) dx = L/2 δₘₙ
```
