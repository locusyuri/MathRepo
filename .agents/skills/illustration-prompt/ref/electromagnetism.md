# Electromagnetism Illustration Templates（电磁学插图模板）

本子技能涵盖静电学与静磁学的插图模板，用于生成电场和磁场的模型示意图与函数曲线图。

---

## 电场与磁场的本质差异

在编写电场或磁场模型提示词之前，必须理解以下本质区别：

| 对比项 | **电场（点电荷）** | **磁场（载流直导线）** |
|-------|------------------|---------------------|
| 场线形态 | **径向辐射**（从正电荷发出→负电荷终止） | **同心闭合圆环**（环绕导线，无始无终） |
| 场线方向 | 正电荷向外 / 指向负电荷 | 右手定则（电流向上 → 逆时针环绕） |
| 衰减规律 | $E \propto 1/r^2$（反平方） | $B \propto 1/\rho$（反比，衰减更慢） |
| 标量势 | $V$ 存在（$E = -\nabla V$） | **无标量势**（在电流区 $\nabla\times B \neq 0$） |
| 右面板内容 | $E$-$r$ + $V$-$r$ 两条曲线 | **只有 $B$ 一条曲线**（或 $B$ + $A$ 两条曲线） |
| 一致性检查 | $E = -dV/dr$ | $\oint B\cdot dl = \mu_0 I_\text{enc}$ |

## 配色规范

| 元素 | 颜色 | 用途 |
|------|------|------|
| 正电荷/源 | 红色 | 点电荷、从正电荷出发的电场线 |
| 负电荷/汇 | 蓝色 | 负电荷、指向负电荷的电场线 |
| 电场强度 E | 蓝色 | 电场矢量、E 曲线 |
| 磁场 B | 绿色 | 磁场矢量、B 曲线 |
| 电势 V | 绿色或橙色 | V 曲线 |
| 矢势 A | 橙色 | A 曲线 |
| 电流/导线 | 红色 | 载流导线、电流方向箭头 |
| 网格/渐近线 | 浅灰色 | 辅助线 |
| 中性元素 | 黑色 | 坐标轴、标签、文字、图框 |

## 磁场的"3D vs 2D"视角选择

磁场模型图需要特别关注视角选择，因为磁感线是空间的闭合环：

| 视角 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **3D 侧视图**（导线竖直，磁感线环投影为椭圆） | 无限长直导线、同轴电缆等 | 同时显示导线和环绕的磁场线，最直观 | 椭圆可能被误认为磁场本身形状 |
| **2D 顶视图**（沿导线轴向看） | 纯磁场线拓扑示意 | 显示真正的圆环形 | 无法显示导线和电流方向 |
| **3D 轴测图** | 有限长导线、电流环 | 立体感强，适合演示 | 结构较复杂 |

**推荐默认**：**3D 侧视图**，磁场线为环绕导线的同心圆环（投影为椭圆），B 矢量沿环的切向。

## 电场模型模板

### 点电荷电场

**模型图**（左面板）：
- 红色实心圆点 `+q`，位于画面中央
- 12 条红色径向电场线向外辐射，带箭头
- 若干测试点处的蓝色 E 矢量箭头
- 图例

**函数图**（右面板，合并 E-r 和 V-r）：
- 同一坐标系中画两条曲线，用不同颜色区分
- 蓝色曲线：$E = kq/r^2$（用左侧 y 轴）
- 绿色/橙色曲线：$V = kq/r$（用右侧 y 轴；若量级相近也可共用同一 y 轴）
- 在 $r \to \infty$ 处画水平虚线渐近线 $V \to 0$
- 标注关键数值点
- 选取范围 $r \in [0.5, 4]$ 体现从快速衰减到趋于平缓的全过程
- 不标注具体参数值（k、q 等）

### 均匀带电球壳

**函数图**（右面板，合并 E-r 和 V-r）：
- 蓝色曲线：$E(r)$ 分段函数
  - $r < R$: $E = 0$（水平线）
  - $r \ge R$: $E = kQ/r^2$（递减曲线）
- 绿色曲线：$V(r)$ 分段函数
  - $r \le R$: $V = kQ/R$（水平线）
  - $r \ge R$: $V = kQ/r$（递减曲线）
- 在 $r = R$ 处画竖直虚线标注分界点
- 覆盖 $r \in [0, 4R]$ 展示全部三段行为

### 均匀带电球体

**函数图**（右面板，合并 E-r 和 V-r）：
- 蓝色曲线：$E(r)$
  - $r < R$: $E \propto r$（线性增长）
  - $r \ge R$: $E = kQ/r^2$（递减）
- 绿色曲线：$V(r)$
  - $r \le R$: $V = \dfrac{kQ}{2R}\left(3 - \dfrac{r^2}{R^2}\right)$（抛物线）
  - $r \ge R$: $V = kQ/r$（递减）
- 在 $r = R$ 处画竖直虚线
- 覆盖 $r \in [0, 4R]$

### 平行板电容器

**模型图**（左面板）：
- 两块平行板，带 `+Q` 和 `-Q`
- 板间均匀电场线（平行等距蓝色箭头），板外无场

**函数图**（右面板，合并 E-x 和 V-x）：
- 蓝色曲线：$E(x)$ — 板间为常数 $E = \sigma/\epsilon_0$，板外为 0
- 绿色曲线：$V(x)$ — 板间线性下降，板外常数
- 覆盖 $x$ 从一块板外侧到另一块板外侧的全范围

### 电偶极子

**模型图**（左面板）：
- 一对 `+q`（红色）和 `-q`（蓝色），间距 $d$
- 电场线从正电荷出发到负电荷终止（弯曲的弧线）
- 标注偶极矩 $\mathbf{p} = q\mathbf{d}$

**函数图**（右面板）：
- 画一条沿偶极子轴线的 $E(x)$ 或 $V(x)$ 曲线
- 标注 $x \to \pm\infty$ 时场强趋于 0

## 磁场模型模板

### 无限长直导线的磁场

**模型图**（左面板，3D 侧视图）：
- 画面中央一条竖直的灰色圆柱体（导线），沿 z 轴放置
- 导线内部一个红色箭头标注电流方向：竖直向上，标签 I
- 环绕导线的 4 个同心圆环（深绿色虚线或实线），半径递增，圆心在导线轴线上
- 每个圆环上带方向箭头（逆时针方向），表示磁感线方向（右手定则：电流向上 → 逆时针环绕）
- 在 ρ 处（右方某点）画一个绿色箭头 B，沿切向（逆时针方向），标签 bold(B)(rho)
- 标注从导线轴心到该点的水平虚线，带双向箭头和标签 rho
- 图例：红色箭头 = Current I，绿色圆环 = Magnetic field lines，绿色箭头 = B
- 画面简洁，突出"环绕"和"闭合"的磁场特征
- **说明**：这是 3D 侧视图，导线竖直，磁场线是环绕导线的同心圆环（投影为椭圆），B 矢量沿切向。
- **导线表示**：用灰色圆柱体（非纯直线）是为了体现 3D 立体感，避免被误认为 2D 平面内的线段。圆柱体的粗细应适中，不遮挡磁场线。

**函数图**（右面板）：
- 横轴：Radial Distance ρ (m)，范围 [0.5, 5]
- 纵轴：Magnetic Field B(ρ) (T)
- 绿色实线曲线：B(ρ) = μ₀I / (2πρ)，从 ρ=0.5 到 ρ=5
- 在曲线旁标注公式：B = μ₀I / (2πρ)（粗体，绿色）
- 标注 2-3 个特征点
- ρ → ∞ 处画水平虚线渐近线 B → 0
- 浅灰色网格线（虚线），适当密度
- 图例：绿色实线 = B(ρ) ∝ 1/ρ

**说明**：该模型没有标量势 V，因此右面板只画 B(ρ) 单条曲线。磁场随距离衰减比点电荷电场慢（1/ρ vs 1/ρ²）。

### 均匀载流圆柱体

**模型图**（左面板）：
- 灰色圆柱体（半径 R），内部均匀电流密度 J
- 圆柱体内部红色箭头表示电流方向（沿 z 轴）
- 圆柱体外部若干个同心圆环（绿色磁感线）
- 标注 R、J、I = JπR²

**函数图**（右面板，合并 B-ρ 和 A-ρ）：
- 横轴：Radial Distance ρ (m)，范围 [0, 4R]
- 左侧纵轴：Magnetic Field B(ρ) (T)
- 右侧纵轴：Vector Potential A_z(ρ) (Wb/m)
- 绿色曲线：B(ρ) 分段函数
  - ρ < R: B = μ₀Jρ/2（线性增长）
  - ρ ≥ R: B = μ₀I/(2πρ)（反比衰减）
- 橙色曲线：A_z(ρ) 分段函数
  - ρ < R: A_z = μ₀J(R²-ρ²)/4（抛物线）
  - ρ ≥ R: A_z = μ₀I/(2π) ln(ρ₀/ρ)（对数衰减）
- 在 ρ = R 处画竖直虚线标注分界点
- 图例：绿色 = B(ρ)，橙色 = A_z(ρ)

## 电磁学模型的一致性检查

### 电场模型

```
- E = -dV/dr: 验证数值微分
- 高斯定律：∮E·dS = Q_enc/ε₀
- 边界条件：E 在表面连续/不连续
```

### 磁场模型

```
- 安培环路定理：∮B·dl = μ₀I_enc
  验证：B(ρ)·2πρ = (μ₀I/(2πρ))·2πρ = μ₀I ✓
- 高斯磁定律：∇·B = 0（磁感线闭合）✓
- 矢势关系：B = ∇×A（在库仑规范下）✓
- 对比静电场：B(ρ) = μ₀I/(2πρ) ↔ E(ρ) = λ/(2πε₀ρ)（数学形式完全平行）✓
```
