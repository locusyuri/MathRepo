\chapter{Plane Separation Theorem} % 平面分割定理

\begin{definition}{Jordan Curve and Simple Closed Curve}\label{def:jordan_curve} % 简单曲线与简单闭曲线
    Let \(X\) be a topological space.
    If there exists a continuous injective mapping \(\gamma: [0, 1] \to X\), 
    then the image \(\gamma([0, 1])\) is called a \textbf{Jordan curve} or \textbf{simple closed curve} in \(X\). 

    % 如果还满足 \(\gamma(0) = \gamma(1)\), 则称 \(\gamma([0, 1])\) 是 \(X\) 中的一条 \textbf{简单闭曲线}。
    If \(\gamma(0) = \gamma(1)\), then \(\gamma([0, 1])\) is called a \textbf{simple closed curve} in \(X\).

    % 我们还有另一种定义: 简单闭曲线是同胚于单位圆周 \(S^1\) 的空间
    Another definition is that a simple closed curve is a space homeomorphic to the unit circle \(S^1\).
\end{definition}
